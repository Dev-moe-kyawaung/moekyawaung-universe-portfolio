## 🐛 Known Issues & Solutions

### Issue: Particles Cause Lag on Low-End Devices

**Solution:**
```javascript
// Already implemented auto-detection
// Manual adjustment:
const particleCount = isMobile ? 75 : 150;
```

### Issue: Starfield Not Visible

**Cause:** Background too bright or stars not generating

**Solution:**
```javascript
// Check initialization
console.log('Starfield stars:', document.querySelectorAll('.star').length);
// Should show 200 stars
```

### Issue: Vortex Rings Flickering

**Solution:**
```css
.vortex-ring {
    will-change: transform;
    backface-visibility: hidden;
    transform: translateZ(0); /* Force GPU */
}
```

### Issue: Rainbow Gradient Not Animating

**Solution:**
```css
.element {
    background-size: 200% auto; /* Must be > 100% */
    animation: rainbowShift 5s linear infinite;
}
```
